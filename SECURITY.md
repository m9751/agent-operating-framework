# Security Policy

## Scope

This repository contains prose documents only — framework rules, guides, and shell script examples. There is no application code, no server, and no dependency tree.

Security reports about:
- **Secrets accidentally committed** — yes, please report
- **Shell script logic errors** in `examples/hooks/` that could cause unintended behavior — yes, please report
- **Broken links or malformed content** — open a regular issue instead
